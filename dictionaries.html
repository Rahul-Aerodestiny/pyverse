Python Dictionaries

Learn how to store data in key-value pairs and access them efficiently.

Try it Yourself →

What is a Dictionary?

A dictionary in Python is a collection of key-value pairs. It’s unordered (before Python 3.7), mutable, and optimized for fast lookups. 

student = {
  "name": "Rahul",
  "age": 28,
  "course": "Python"
}
print(student)
print(student["name"])    # Rahul

Accessing and Modifying Dictionaries

You can access values using their keys, add new items, or modify existing ones. 

student["age"] = 29         # Modify value
student["city"] = "Pune"    # Add new key-value pair

print(student)

# Use get() to avoid KeyError
print(student.get("phone", "Not Found"))

Common Dictionary Methods

person = {"name": "Deepanjali", "age": 27, "city": "Pune"}

print(person.keys())      # dict_keys(['name', 'age', 'city'])
print(person.values())    # dict_values(['Deepanjali', 27, 'Pune'])
print(person.items())     # dict_items([('name', 'Deepanjali'), ('age', 27), ('city', 'Pune')])

# Remove an item
person.pop("city")
print(person)

# Clear all items
person.clear()

Looping Through a Dictionary

You can iterate over keys, values, or both using loops.

for key, value in student.items():
    print(key, ":", value)
