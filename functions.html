Python Functions

Functions let you organize your code into reusable blocks that make it cleaner, modular, and easy to maintain.

Try it Yourself →

What are Functions?

A function is a block of code that runs only when it is called. You can pass data into a function using parameters and return results using return. 

# Defining and Calling a Function
def greet():
    print("Hello from PyVerse!")

greet()  # Calling the function

Function Parameters

Functions can take arguments that make them dynamic and flexible.

def greet_user(name):
    print(f"Hello, {name}!")

greet_user("Rahul")
greet_user("Aisha")

Return Values

Functions can send back data using the return statement. Returned values can be stored in variables or used directly. 

def add(a, b):
    return a + b

result = add(5, 7)
print("Sum:", result)

Default and Keyword Arguments

You can set default values for parameters and also use keyword arguments for clarity. 

def introduce(name, country="India"):
    print(f"I am {name} from {country}.")

introduce("Rahul")
introduce("Emily", "Canada")

Lambda (Anonymous) Functions

Lambda functions are small, one-line anonymous functions used for quick operations. 

square = lambda x: x ** 2
print(square(5))   # Output: 25

add = lambda a, b: a + b
print(add(3, 4))   # Output: 7
