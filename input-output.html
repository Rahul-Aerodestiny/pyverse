Python Input and Output

Learn how to take input from users and display output in Python programs — the very foundation of interactivity.

1. Taking User Input

In Python, you can use the input() function to accept data from the user. It always returns the input as a string, so you often need to convert it to the desired type.

Example: 
name = input("Enter your name: ")
print("Hello,", name)

2. Type Conversion for Inputs

Since input() returns a string, you can convert it to integers or floats using int() or float(). 

x = int(input("Enter first number: "))
y = int(input("Enter second number: "))
print("Sum is:", x + y)

3. Printing Output

The print() function displays output on the screen. You can print multiple items separated by commas, or customize the separator and end character.

Examples: 
print("Hello World!")
print("Python", "is", "fun")
print("Welcome", end=" ")
print("to PyVerse!")

4. String Formatting

Python allows you to format output neatly using f-strings or the format() method.

name = "Rahul"
age = 30
print(f"My name is {name} and I am {age} years old.")

5. Reading Multiple Inputs

You can take multiple inputs in one line using split().

a, b, c = input("Enter three numbers: ").split()
print("Numbers are:", a, b, c)

6. File Input and Output

Python also allows reading and writing to files using open(), read(), and write() methods.

# Writing to a file
with open("demo.txt", "w") as f:
    f.write("Hello, PyVerse!")

# Reading from a file
with open("demo.txt", "r") as f:
    print(f.read())
