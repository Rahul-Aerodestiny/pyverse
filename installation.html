Python Installation & Setup

Step 1: Install Python

- Download Python from the official site: python.org/downloads
- During installation, check "Add Python to PATH". - Verify installation in terminal: python --version or python3 --version

Step 2: Setup Virtual Environment

Virtual environments help you isolate dependencies for different projects. 


# Install virtualenv if not installed
pip install virtualenv

# Create virtual environment
python -m venv myenv

# Activate virtual environment
# On Windows
myenv\Scripts\activate
# On Mac/Linux
source myenv/bin/activate

# Deactivate environment
deactivate
