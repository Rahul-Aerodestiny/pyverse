More on Python Lists

Take your list knowledge to the next level with loops, comprehensions, and methods.

Iterating with For Loops

Loop through elements directly or use enumerate() for index + value.

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

for i, fruit in enumerate(fruits):
    print(i, fruit)

List Comprehension

Concise way to generate lists, often replacing loops.

squares = [x**2 for x in range(5)]
evens = [x for x in range(10) if x % 2 == 0]
matrix = [[i*j for j in range(3)] for i in range(3)]

Common List Methods

Python lists have many built-in methods for adding, removing, and manipulating elements.

nums = [3, 1, 4]
nums.append(2)
nums.extend([5, 6])
nums.insert(1, 10)
nums.remove(4)
last = nums.pop()
nums.sort()
nums.reverse()

Copying Lists

Assigning a list to another variable doesn’t copy it — both point to the same object.

a = [1, 2, 3]
b = a           # reference, not copy
b.append(4)
print(a)        # [1, 2, 3, 4]

# Correct ways to copy
c = a.copy()
d = list(a)
e = a[:]

Shallow vs Deep Copy

Nested lists need copy.deepcopy() for a true copy.

            
import copy
nested = [[1, 2], [3, 4]]
shallow = nested.copy()
deep = copy.deepcopy(nested)

shallow[0][0] = 99
print(nested)   # [[99, 2], [3, 4]]
print(deep)     # [[1, 2], [3, 4]]
