Python Lists

Master ordered, mutable collections with practical examples.

What are Lists?

Lists in Python are ordered, mutable collections that can store multiple elements, including different data types. They are one of the most powerful and commonly used data structures.

fruits = ["apple", "banana", "cherry"]
print(fruits[0])   # Output: apple
print(len(fruits)) # Output: 3

Creating Lists

Lists can be created using square brackets [] or the list() constructor.

numbers = [1, 2, 3, 4]
mixed = [1, "hello", 3.5, True]
empty = list()

Indexing & Slicing

Lists support zero-based indexing and slicing like strings.

fruits = ["apple", "banana", "cherry", "date"]
print(fruits[1])     # banana
print(fruits[-1])    # date
print(fruits[1:3])   # ['banana', 'cherry']

Iterating over Lists

for fruit in fruits:
    print(fruit)

for i, fruit in enumerate(fruits):
    print(i, fruit)

Adding & Removing

fruits.append("kiwi")
fruits.insert(1, "mango")
fruits.remove("banana")
last = fruits.pop()
fruits.clear()

Sorting & Reversing

numbers = [4, 2, 9, 1]
numbers.sort()         # [1, 2, 4, 9]
numbers.sort(reverse=True)
numbers.reverse()

Nested Lists

matrix = [[1,2,3], [4,5,6], [7,8,9]]
print(matrix[1][2])   # 6

List Comprehensions

Concise way to create lists using loops and conditions.

squares = [x**2 for x in range(5)]
evens = [x for x in range(10) if x % 2 == 0]

Built-in Functions

numbers = [4, 2, 9, 1]
print(len(numbers))  # 4
print(min(numbers))  # 1
print(max(numbers))  # 9
print(sum(numbers))  # 16

Copying Lists

a = [1, 2, 3]
b = a.copy()
c = list(a)
d = a[:]

Common Pitfalls

Be careful when assigning lists directly — both variables refer to the same object.

a = [1, 2, 3]
b = a
b.append(4)
print(a)  # [1, 2, 3, 4] (changed!)
