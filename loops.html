Python Loops

Loops help you execute a block of code multiple times efficiently.

Try it Yourself →

For Loops

for i in range(5):
    print("Iteration", i)

Repeats the block 5 times with i values from 0 to 4.

While Loops

count = 0
while count < 5:
    print("Count is", count)
    count += 1

Executes until the condition becomes False.

Loop Control Statements

  • break: Exit the loop immediately
  • continue: Skip the current iteration
  • pass: Placeholder, does nothing
