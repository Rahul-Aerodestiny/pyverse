Modules in Python

Reuse, organize, and share your code effectively using Python modules and packages.

Try it Yourself →

What is a Module?

A module is a Python file (.py) containing reusable code — functions, classes, or variables — that can be imported into other scripts. A package is a collection of modules, typically organized in a folder with an __init__.py file.

Importing Modules

# import the whole module
import math
print(math.sqrt(16))
print(math.pi)

# import specific items
from math import sqrt, pi
print(sqrt(25))

Aliasing & Best Practices

# aliasing
import numpy as np

# avoid * imports — it can overwrite existing names
from math import *  # not recommended

Common Built-in Modules

import os        # for file & directory operations
import sys       # system-specific parameters & paths
import json      # JSON encoding/decoding
import datetime  # working with dates and times
import random    # random number generation
import itertools # advanced iteration tools

Third-Party Packages using pip

Install packages from PyPI (Python Package Index) using pip.

# install (in terminal)
pip install requests

# use it in your script
import requests
response = requests.get("https://api.github.com")
print(response.status_code)

__name__ == "__main__" Trick

This allows your file to act both as a standalone script and an importable module. 

# mymodule.py
def greet(name):
    print("Hello", name)

if __name__ == "__main__":
    greet("Rahul")  # runs only if executed directly

Organizing Code into Packages

mypackage/
    __init__.py
    utils.py
    helpers.py

# import from package
from mypackage.utils import greet

Understanding sys.path

Python searches for modules in the directories listed in sys.path.

import sys
print(sys.path)   # shows all directories Python checks for modules

Creating a Package for Distribution

To share your module or library with others, define metadata using a pyproject.toml file. Here’s a minimal setup: 

# pyproject.toml
[build-system]
requires = ["setuptools"]
build-backend = "setuptools.build_meta"

[project]
name = "mypackage"
version = "0.1.0"
authors = [{ name = "Rahul Kumar", email = "rahul@example.com" }]
description = "A small Python utility package"
readme = "README.md"
requires-python = ">=3.8"
license = {text = "MIT"}
dependencies = ["requests"]

# After writing this file:
# python -m build
# pip install dist/mypackage-0.1.0-py3-none-any.whl
