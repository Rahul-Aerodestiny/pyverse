Python Sets

Learn about unordered, unique collections in Python and perform powerful set operations.

Try it Yourself →

What are Sets?

A set is an unordered collection of unique elements. It’s useful for removing duplicates and performing mathematical operations like union, intersection, and difference. 

fruits = {"apple", "banana", "cherry", "apple"}
print(fruits)   # {'banana', 'cherry', 'apple'} — duplicates removed

# Create empty set
empty_set = set()
print(type(empty_set))    #

Set Operations

Sets support operations similar to mathematical sets.

A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

print(A | B)    # Union → {1, 2, 3, 4, 5, 6}
print(A & B)    # Intersection → {3, 4}
print(A - B)    # Difference → {1, 2}
print(A ^ B)    # Symmetric difference → {1, 2, 5, 6}

Common Set Methods

colors = {"red", "green", "blue"}

colors.add("yellow")      # Add element
colors.remove("green")    # Remove element (error if not present)
colors.discard("purple")  # Remove if present, else ignore
colors.clear()            # Empty the set

print(colors)

Set Comprehension

Like list comprehensions, you can create sets with concise syntax.

nums = {x**2 for x in range(6)}
print(nums)   # {0, 1, 4, 9, 16, 25}
