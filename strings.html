Python Strings

Learn how to work with text, manipulate it, and use powerful string methods.

Try it Yourself →

String Basics

name = "Rahul"
greeting = "Hello, " + name
print(greeting)  # Output: Hello, Rahul

Common String Methods

text = "python programming"
print(text.upper())   # PYTHON PROGRAMMING
print(text.lower())   # python programming
print(text.replace("python", "Java"))  # Java programming
print(text.split())   # ['python', 'programming']

String Slicing

text = "Python"
print(text[0])    # P
print(text[:4])   # Pyth
print(text[2:])   # thon
print(text[-1])   # n
