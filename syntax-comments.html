Python Syntax & Comments

Understanding the basic syntax and writing clean, well-documented code is the foundation of Python programming.

Python Syntax Overview

Python syntax defines the set of rules that describe how a Python program is written and interpreted. Python emphasizes simplicity and readability.

Example: Simple Python Code 
print("Welcome to PyVerse!")
x = 5
y = 10
print(x + y)

Key Points:

  • No need for semicolons at the end of statements.
  • Code blocks are defined by indentation (spaces), not braces {}.
  • Python uses 4 spaces per indentation level (not tabs).
  • Case sensitivity matters — MyVar and myvar are different.

Indentation in Python

Indentation is used to define the scope of loops, functions, classes, and conditional statements.

Example: 
if 5 > 2:
    print("Five is greater than two!")  # Correct indentation

# Incorrect indentation example
if 5 > 2:
print("Five is greater than two!")  # ❌ This will cause an error

Comments in Python

Comments are used to explain code and make it more readable. They are ignored by the Python interpreter.

Single-line Comment 
# This is a comment
print("Hello, World!")  # This prints a message
Multi-line Comment 
"""
This is a multi-line comment
spanning multiple lines.
"""
print("Comments are ignored by Python!")

✅ Pro Tip: Use meaningful comments to describe why a piece of code exists — not just what it does.

Example: Clean & Commented Code

# Program to find the largest number
numbers = [15, 42, 7, 23, 89, 3]

# Initialize max_num with the first element
max_num = numbers[0]

# Loop through all numbers to find the maximum
for num in numbers:
    if num > max_num:
        max_num = num

print("The largest number is:", max_num)
