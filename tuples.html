Python Tuples

Learn how to use immutable, ordered collections in Python to store multiple values.

Try it Yourself →

What are Tuples?

A tuple is an ordered, immutable collection in Python. Unlike lists, tuples cannot be modified once created. They are useful for storing fixed sets of data, like coordinates or database records. 

my_tuple = (10, 20, 30)
print(my_tuple)

# Access elements
print(my_tuple[0])   # 10

# Nested tuple
nested = (1, 2, (3, 4))
print(nested[2][1])  # 4

Tuple Methods

Tuples support only a few methods since they are immutable:

nums = (1, 2, 3, 2, 1, 2)
print(nums.count(2))   # 3 (number of times 2 appears)
print(nums.index(3))   # 2 (position of first 3)

Tuple Immutability

You cannot change, add, or remove items in a tuple after it’s created. However, tuples can contain mutable objects like lists. 

data = (1, [2, 3], 4)
data[1].append(5)
print(data)  # (1, [2, 3, 5], 4)
