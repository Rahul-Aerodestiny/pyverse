Python Variables

Learn how to store, modify, and manage data effectively in Python

What Are Variables?

Variables are containers for storing data values. In Python, you don’t need to declare them explicitly — just assign a value, and Python creates the variable automatically.

# Example
name = "Rahul"
age = 28
is_student = False
print(name, age, is_student)

Naming Rules

  • Must start with a letter or underscore (_).
  • Cannot start with a number.
  • Case-sensitive (MyVar ≠ myvar).
  • Use descriptive names like student_name.
Pro Tip: Use snake_case for variable names.

Dynamic Typing

Python allows you to change the data type of a variable by simply reassigning a new value.

x = 100
print(x, type(x))

x = "Python"
print(x, type(x))

Multiple Assignment

Assign values to multiple variables in one line.

a, b, c = 5, 10, 15
print(a, b, c)

Constants

Python doesn’t have true constants. By convention, constants are written in uppercase.

PI = 3.14159
GRAVITY = 9.8
print(PI, GRAVITY)

Variable Scope

Scope defines where a variable can be accessed. Python has two types of scope:

  • Global: Declared outside functions.
  • Local: Declared inside functions.
x = 50  # Global

def demo():
    x = 10  # Local
    print("Inside function:", x)

demo()
print("Outside function:", x)

Using the global Keyword

Use global when you need to modify a global variable inside a function.

count = 0

def increment():
    global count
    count += 1

increment()
print(count)

Deleting Variables

Use del to remove variables from memory.

name = "Python"
print(name)
del name
# print(name)  # This will raise NameError
